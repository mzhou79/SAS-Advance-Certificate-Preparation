# SAS Advance Certificate Preparation
Core to pass the "SAS Certified Professional: Advanced Programming Using SAS 9.4" exam. I recommand to follow those three steps and you will have a 99\% probability to pass:

### 1. Read the textbook (and complete all the questions after each unit).

### 2. Lab
practice the 20 questions in the lab file (dataset provided) until you can write them with your eyes closed. In the real exam, all the lab questions will be 95\% same with those 20 questions (basically choose 11 out of 20 and nothing surprise). Only pay attention to minor details, like how to hide the output, keep selected variables, etc.

The lab file contains generated dataset you will use. Paste them into your SAS On Demand for Academics and run them before you answer the problems.
